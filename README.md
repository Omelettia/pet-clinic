# Pet Clinic - ITSS - G10
## Setup
- Use init.sql file in psql-docker to create postgresql database
- Change information in connect.php in utils directory (host,db,user,password)
- run index.php file to get to the home
## Technologies 
- PHP
- Javascript
- PostgreSQL
